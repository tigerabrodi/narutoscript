# narutoscript

To install dependencies:

```bash
bun install
```

To run a file:

```bash
bun run src/index.ts examples/hello.naru
```

To start the REPL:

```bash
bun run src/index.ts
```

More examples:

```bash
bun run src/index.ts examples/ninja-power-calculator.naru
bun run src/index.ts examples/all-features.naru
```
